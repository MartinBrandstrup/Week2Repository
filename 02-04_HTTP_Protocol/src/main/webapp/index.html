Ex 2:

Kode 304 på image og kode 200 på css stylesheet

Jeg antager at det er den her connection header:

Accept-Language: da-DK,da;q=0.9,en-US;q=0.8,en;q=0.7

Connection: keep-alive

Host: localhost:8080

Den bestemmer hvilken type connection der er tale om, og hvor længe den skal vare. Keep-alive betyde den er persistent og kan modtage flere requests.
